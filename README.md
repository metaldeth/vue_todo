# Vue Todo

Учебный проект: учим Vue 3, делая классический Todo List своими руками.

## Требования

- **Node.js** 18+ (проверка: `node -v`)
- **npm** (идёт с Node.js): `npm -v`

Vue **не нужно** ставить глобально — всё ставится в папку проекта через `npm install`.

## Как запустить

1. Открой терминал и перейди в папку проекта:
   ```bash
   cd c:\dev\vue_todo
   ```
2. Установи зависимости (один раз):
   ```bash
   npm install
   ```
3. Запусти сервер:
   ```bash
   npm run dev
   ```
4. Открой в браузере адрес из вывода (обычно **http://localhost:5173**).

## Если не запускается

- **«npm не является внутренней командой»** — Node.js не установлен или не в PATH. Установи [Node.js](https://nodejs.org/) (LTS) и перезапусти терминал.
- **Ошибка при `npm install`** — проверь, что ты в папке, где лежит `package.json` (корень проекта).
- **Ошибка при `npm run dev`** — убедись, что перед этим выполнен `npm install`. Если порт 5173 занят, Vite выведет другой порт (например 5174) — открой его в браузере.
- **Страница пустая или ошибка в браузере** — открой DevTools (F12) → вкладка Console и посмотри текст ошибки.

## Обучение

Все пошаговые инструкции — в файле **[LEARNING.md](./LEARNING.md)**. Открой его и выполняй задания по порядку; Todo List нужно реализовать самому по этому руководству.
